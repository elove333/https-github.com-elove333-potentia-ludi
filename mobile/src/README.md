# Source Code Directory

This directory contains the main application source code.

## Structure

- **screens/** - UI screens and views
- **components/** - Reusable UI components
- **navigation/** - Navigation configuration and routing
- **services/** - Business logic and external integrations
    - **motion-tracking/** - Camera and sensor integration for motion detection
    - **ai-models/** - AI/ML models for movement analysis
    - **api/** - Backend API communication
    - **audio/** - Audio playback and rhythm management
- **hooks/** - Custom React hooks
- **utils/** - Helper functions and utilities
- **assets/** - Static resources (images, fonts, sounds)
- **styles/** - Global styles and theme configuration
