# Services

This directory contains service modules that handle business logic and external integrations.

## Subdirectories

**motion-tracking/**

Handles camera access, sensor data, and motion detection logic.

**ai-models/**

Contains AI/ML model integrations for analyzing user movements and providing feedback.

**api/**

Backend API communication, including user data, progress tracking, and authentication.

**audio/**

Audio playback management, rhythm synchronization, and sound effects.
